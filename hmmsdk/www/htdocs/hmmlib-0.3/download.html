HmmLib: Hidden Markov Model Library in Java

License Agreement

Even if this software is available for free, it is nontheless copyrighted. Please view the license before you download the software.

Download Complete Package

Most people simply download the Complete Package, which includes both source programs and documentations in addition to the compiled classes (in jar file).

Download Source Codes

If you are only interested in viewing the source codes, or if you intend to compile them yourself, simply download the Source Codes, which is only 19k in size. You need Java Platform 2 Development Kit (JDK2) in order to be able to compile this library yourself.

Download Documentations

If you only need to see the API specifications, download the Documentations.

Download Class Files

You can also download only the byte-compiled class files of HmmLib v0.3, hmmlib.jar. Do not unzip this jar file. Put this archive into your classpath.

Download Information on XML Parsers in Java

You also need the following XML parsers in order to be able to use HmmLib. Please note the alpha status for both programs. Also note that you need to accept their copyright agreements before you can use these softwares!

Sun's XML parser early-access release
Please visit Sun's JDC early access home page. For your convenience, you can also download the jar file, xml.jar from this site. (Note that this is not a complete package.)
IBM's XML parser, XML4J
Please visit IBM's alphaworks home page. For your convenience, you can also download the jar file, xml4j_1_1_9.jar from this site. (Note that this is not a complete package.)
