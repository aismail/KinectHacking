HmmLib: Hidden Markov Model Library in Java

Welcome to the Home-Site of HmmLib

>> Feb. 6th, 2003    HmmLib home page has been moved to SourceForge.net!

HmmLib is a Java class library which contains all essential functionalities of hidden Markov models. It is not intended to be used for commercial purposes, for the efficiency was not the primary factor in its implementation. Rather, it is hoped that the implementation is clear and transparent enough that it can be used as a model program for students wishing to learn basics of HMMs. Please start from What's New? for latest information on HmmLib. Some of the most important updates in the most recent release are summarized in Release Note.

Description

HmmLib contains two essential classes and some utility classes. Model.java is a core class which stores all model parameters relevant to the underlying Markov model, whereas Hmm.java implements all essential functions of a hidden Markov model, including Forward-backward, Viterbi, and Baum-Welch algorithms.

The current release does not contain any example classes. But, main functions of each class illustrate their basic usages. For more information, please see the API documentation, or browse the source codes on line.

System Requirements

You need to have Java Platform 2 Development Kit (JDK2) in order to be able to use this library. JDK2 is available only on Windows and Solaris as of this writing. Please visit Sun's Java Site for further information.

XML parsers of Sun and IBM are also needed from HmmLib v 0.3, since it used XML as its internal file format to save HMM model parameters. For more information, please see the Download Page.
