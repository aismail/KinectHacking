HmmLib: Hidden Markov Model Library in Java
$Id: readme.html,v 1.5 2003/05/28 04:57:47 drwye Exp $

HmmLib is a Java class library which contains all essential functionalities of hidden Markov models (HMMs). It is not intended to be used for commercial purposes, for the efficiency was not the primary factor in its implementation. Rather, it is hoped that the implementation is clear and transparent enough that it can be used as a model program for students wishing to learn basics of HMMs. HmmLib is now a part of a larger software package called HmmSDK. Please visit its home page for more information.

System Requirements

You need the version 1.4 or later of the software development kit for Java 2 Platform Standard Edition (J2SE) in order to be able to use this library. If you have an eariler version of JDK, then you might still be able to use this: You'll need some extra libraries like XML parsers, and you might have to comment out some JDK1.4 specific keywords such as assert. This configuration is, however, not supported, and HmmLib is currently being developed only with JDK 1.4.

Installation

  • Unzip hmmsdk-hmmlib-bin-M.m.r.b.zip into an empty directory. (M.m.r.b represents a version number of a particular release.) 
    unzip hmmsdk-hmmlib-bin-M.m.r.b.zip
  • Please make sure you have the following folders and files:
    bluecraft.hmmlib-M.m.jarClass and resource files. Do not unzip this archive.
    lib/Third-party library directory. These libraries may be required at either run-time or compile-time in order to be able to use HmmLib.
    schema/XML Schema/DTD directory, which includes document type definition files for HMM serialization file format.
    docs/Documentation directory, which includes this readme file and the public license file for this software.
    apidocs/Javadoc directory. HTML-ized version of source files can be accessed from Javadocs.
     
  • Put bluecraft.hmmlib-M.m.jar into your CLASSPATH. (For example, on Windows) 
    set CLASSPATH=%CLASSPATH%;"YOUR_DIRECTORY"\bluecraft.hmmlib-M.m.jar
