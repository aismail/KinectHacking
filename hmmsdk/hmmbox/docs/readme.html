HmmBox: Hidden Markov Model in the Box
$Id: readme.html,v 1.2 2003/05/28 04:57:02 drwye Exp $

HmmBox is a text or console-based test tool. It can also be used as a general-purpose "shell". It is not intended to be used for commercial purposes, for the efficiency was not the primary factor in its implementation. Rather, it is hoped that the implementation is clear and transparent enough that it can be used as a model program for students wishing to learn basics of hidden Markov models (HMMs). HmmBox is a part of a larger software package called HmmSDK, and it is built on top of the Hidden Markov Model Library, HmmLib, which contains all essential functionalities of HMMs. Please visit its home page for more information.

System Requirements

You need the version 1.4 or later of the software development kit for Java 2 Platform Standard Edition (J2SE) in order to be able to use this library. If you have an eariler version of JDK, then you might still be able to use this: You'll need some extra libraries like XML parsers, and you might have to comment out some JDK1.4 specific keywords such as assert. This configuration is, however, not supported, and HmmBox is currently being developed only with JDK 1.4.

Installation

  • Unzip hmmsdk-hmmbox-bin-M.m.r.b.zip into an empty directory. (M.m.r.b represents a version number of a particular release.) 
    unzip hmmsdk-hmmbox-bin-M.m.r.b.zip
  • Please make sure you have the following folders and files:
    bluecraft.hmmbox-M.m.jarClass and resource files. Do not unzip this archive.
    lib/Third-party library directory. These libraries may be required at either run-time or compile-time in order to be able to use HmmBox.
    docs/Documentation directory, which includes this readme file and the public license file for this software.
    apidocs/Javadoc directory. HTML-ized version of source files can be accessed from Javadocs.
     
  • Put bluecraft.hmmbox-M.m.jar and all necessary library files into your CLASSPATH. (For example, on Windows) 
    set CLASSPATH=%CLASSPATH%;"YOUR_DIRECTORY"\bluecraft.hmmbox-M.m.jar
