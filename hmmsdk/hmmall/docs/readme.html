HmmSDK: Hidden Markov Model Development Kit
$Id: readme.html,v 1.1 2003/05/28 04:49:38 drwye Exp $

HmmSDK is a hidden Markov model (HMM) software development kit written in Java. It consists of core library of HMM functions (Forward-backward, Viterbi, and Baum-Welch algorithms) and toolkits for application development. Please visit its home page for more information.

System Requirements

You need the version 1.4 or later of the software development kit for Java 2 Platform Standard Edition (J2SE) in order to be able to use this library. If you have an eariler version of JDK, then you might still be able to use this: You'll need some extra libraries like XML parsers, and you might have to comment out some JDK1.4 specific keywords such as assert. This configuration is, however, not supported, and HmmSDK is currently being developed only with JDK 1.4.

Installation

  • Unzip hmmsdk-full-M.m.r.zip into an empty directory. (M.m.r represents a version number of a particular release.) 
    unzip hmmsdk-full-M.m.r.zip
  • Please make sure you have the following folders (among others):
    hmmlib/Core library module.
    hmmbox/Console applications.
    hmmall/Convenience module to build all sub-modules together.
    hmmall/apidocs/Javadoc directory for all modules. HTML-ized version of source files can be accessed from Javadocs.
     
  • Put hmmall/build/bluecraft.hmmsdk-M.m.jar and all necessary library files into your CLASSPATH in order to use HmmSDK. (For example, on Windows) 
    set CLASSPATH=%CLASSPATH%;"YOUR_DIRECTORY"\bluecraft.hmmsdk-M.m.jar
  • If you would like to build HmmSDK yourself, then take a look at the Ant build script under the hmmall/bin directory. Each module can also be built seperately using its own build script in its bin directory.
